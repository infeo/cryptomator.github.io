2016-12-02

Mysterious Windows Bug Fix with 1.2.3 Update

Cryptomator 1.2.3 for Windows fixes a mysterious bug, commonly known as system error 53 or 67. Well, it’s not that mysterious anymore, but this particular error had been haunting us from the beginning! Cycor on GitHub (almost casually) pointed out in GitHub Issue 210 that he found the cause of system error 53/67 after a few months of experiencing the issue himself. Apparently, there is a Windows Registry entry responsible for listing available network providers called ProviderOrder.
